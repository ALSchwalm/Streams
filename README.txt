C++ STREAMS: Lazy evaluation tools for C++

To use, simply #include "Stream.h", and compile using a C++14 compatible compiler.
All streams classes/functions can be found in the "streams" namespace.

For documentation on streams please see:
    jscheiny.github.io/Streams
